# 博弈策略10：抄底法.因恐慌而犯错

0 每次首阴机会都是拜托大盘暴跌所赐。如果大盘不跌，那是没机会。
1 不是龙头，不考虑。[闽都的技巧.首阴](http://blog.sina.com.cn/s/blog_b46e2fe40102va03.html)
  最好是昨日涨停个股+big.eWolf
2 买点：-8%及tov50%后 + 大红DDE
3 T日，如果权重暴涨而题材股不涨的那一天，尾盘都有机会给你抄底，次日一般都会有不错的利润。
4 T+1日，一般就2种走法，第一种效率高，冲板量创新高收割，不创新高T+2日价格会再新高；第二种要多等一天，冲高收割。
5 T+1日，如果收小阳线的话，后一天会继续反弹。

买点：早盘可抄底，尾盘可抄底，盘中等跌停。

e.g. 中毅达
