# 潜规则1：28原则

太多散户只盯着正确率<胜率>，而忽视了盈亏效率<赔率>。
失败者：做交易对8次错2次还会被腰斩，因为8次是小赚，2次是爆亏。
成功者：做交易对2次错8次还会被腰斩，因为8次是不亏，2次是爆赚。
趋势交易因为<盈亏效率>高，40%胜率也能赚钱。
