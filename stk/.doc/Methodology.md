# 潜规则05：索罗斯.反身性

广义的旁氏骗局：投机者都希望<后来投机者的新入资金>推高而产生收益，而非看好该资产本身能产生的价值。

索罗斯给出的模型提出了<两个离经叛道的观点=投资者行为影响价格与价值(能力)>。

第一个离经叛道的观点是，股票的价格并非围绕股票的价值而波动，而是具有“反身性”。
即“投资者自己的行为本身是影响未来价格的因素”，亦即<过去价格的提升带来升值预期，故而新进入的投资者会买入持有>。

买入持有者并不是看好每股收益而购入，而是他们预期股价会继续上涨，可以卖给后来的投资者。
虽然股票的价值有升值空间，但投机者在这里预期的事情和庞氏骗局是差不多的。即，我所期望的收益是后来的加入者给出的。
所有的人都盼望着后来的加入者加入进来，但参与者的上限，或者说参与者的资金上限并不是无穷的。
这导致整个过程必然面临和旁氏骗局一样的后果——泡沫的破灭。
一旦股价下跌到，光持有股票，获得的股息也是一笔不错的投资时。<套利者>就会出现，或者我们可以叫他们<价值投资者>。
光持有这些廉价的股票，赚取股息即可获得不错的回报率。

买股票的人多了，这公司的融资能力会帮助其扩展业务。下降的时候，会有类似银行挤兑的现象。

TRADE的最大特征：
  1 源自与INDEX的比对，挽救INDEX于危难之际，在其每次低点将其拉起。<引领未来：绝对点数&相对点数>

  3 可持续性基因：<生产力预期=一季报潜伏+>

TRADE的关键概念(点和轨迹)：
  0 <利息理论>：价格波动幅度与交期时长正相关：上午波动巨大&下午越接近<交割点>越平稳。

    每波在两谷间时间长度在20min到50min之间
  3 关键点位处置
    INDEX的关键点位1: 支撑点，即大量筹码成交位置: 未成trough前，使用涨幅的一半:=(high-low)/2
    INDEX的关键点位2: 支撑点，即大量筹码成交位置: 形成trough后，使用trough。
    INDEX的关键点位3: 分时均线，当天平均成交价。

# 博弈策略52：危险信号.当日反转

卖的原因归结于两大类——
第一种是出现危险信号，通常是触发某一事件，需要引起警觉随时准备开溜(仓位温度计)。第二种是趋势走弱，也就是说已经跌到一定的位置我们再出局(渔盆)。

当日反转，《杰西利弗莫尔疯狂的一生》的第十一章是这么解释的(e.g. 男生在做羞羞的事情时，如突然加快速度...)：
(1)如果一天的高点超过前一天的<高点>，(2)但收盘价却比前一天的<收盘价>低，(3)而且<交易量>比前一天大，就会发生当日反转。
反应在盘面上的直观表示，就是股价突然脱离运行轨道，连续快速地暴涨。那么当这波暴涨到尽头出现长上影线或者大阴线的话，就是危险信号，必须要注意。
