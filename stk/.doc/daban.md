# 超短本质

[](http://blog.sina.com.cn/s/blog_69e744a00102uyja.html)
以前，我长期以为超短追求的是短期高收益，经过揣摩学习，我突然领悟到：
<超短其实并不把短期的高收益作为第一追求，而是把长期复利作为第一追求>。
超短并不追求每天能暴利，也不追求资金曲线的快速上涨，而是追求稳定盈利，
追求经过时间的培育能通过复利实现每隔一段时间本金翻倍。

本质上，是<放弃短线的暴利>，而力求每次小赚，控制回撤，积小胜为大胜。

# 口诀 [BAIDU](股票打板族是什么意思?都是什么操作的?)

把握追涨停板时的最佳时机：

1. 大盘好，稀快突突

  (0) 大盘向下时根本不要追涨停。
  (1) 涨停时间越早的比晚的好，早盘先涨停的比尾盘涨停的好得多;
  (2) 涨停时换手率小的比大的好;换手率超过30%的一定不要追。

  (3) 在低位盘整一段时间后突然涨停的比连续上涨后再拉涨停的好;
  (4) 在前期小头部处的放量涨停，一方面说明主力做多意愿坚决，并不惜解放所有的套牢盘其志在高远，另一方面也显示了主力雄厚的资金量和强大的实力。

2. 准备足，不急利众

  (0) 不可忽视操作前的准备工作。
  一是查看有关符合条件个股的股评，第二天的公告信息，股评对大势的预测，对股市有重要影响的事件发展情况。
  有条件的还要看看龙虎榜信息，认真分析主力资金是在流入还是流出，主力资金是机构还是私摹。
  第二天早上，必须认真查看财经消息，看看有没有突发事件、利空消息，顺便回忆一下昨天晚上看过的信息，做好心理准备。

  (1) 失败中总结出来的教训，一定要牢记。
  追涨停的介入点一定要在待涨停个股最后一分钱价位快被消化殆尽，抢在主力的超级大封单出现之前排队，一般都有希望成交，而且这个点位最安全。

  (2) 有重大利好被披露涨停的股票不要追。一般情况下，主力总是对上市公司的利好具有先知先觉的能力。
  不碰连续一字板，开板可能即是跌停。

  (4) 选股有强势原则思路(不从众)
  一是在技术上寻找<处于“两极”>的股票，处于上升趋势加速段的极强势股和远离套牢区、处于超跌中的极弱势股；二是寻找基本面变化对股价构成重大影响的股票。
  精通基本面变化与短期股价的关系，提前研究当天出现的各种重大题材，对题材可操作性排序后确定攻击对象。
  问题股及ST、PT股绝不介入，所选择个股往往业绩尚可，或有一定题材支持，具备成为短期热点的一些必要条件。
  在股票勉强涨停，<但是不封死>，时不时被打开，给人感觉很无力。但这可能是反向洗筹。

# 假机构模式

假机构模式是目前所有短线打板族里面成功率最高的模式，没有之一。因为他解决了打板客的2个核心问题：
1、当天打板的个股能封住，因为假机构模式借鉴了以前徐翔吃独食的模式，利用资金的优势强封，当天封板的个股90%是都会封住的，最近一段时间来他就在荣安地产上被砸了，那时位置高的原因。
2、解决了第二天出货有跟风盘的问题，因为假机构借助了龙虎榜的规则，晚上是一定会出龙虎榜的，这样就可以为第二天带来大量的跟风盘（因为知道是假机构的人还是少数，全国股民这么大，很多资金喜欢跟风机构专用），第二天早盘在成交量充沛的情况下假机构可以顺利出货，还都是大利。假机构模式解决了这两个打板客的核心问题，所向披靡，无人企及，手下的冤魂不计其数。

# 龙兴股的基本特征
从以往的行情发展分析.凡是能够发动一轮富有力度上攻行情的龙头板块，往往其有以下墓本特征:

①龙头股其有一定市场号召力和资金凝服力，并且能有效激发和带动市场人气;
②龙头股行情具有向纵深发展的动能和可持续上涨的潜力;
③龙头股是一渡上升行情里最先启动的股果:
④龙头股是一波上升行情里涨幅最大的股票;
⑤运作龙头股的主力资金实力非常雄厚，并且对欢策面、消色面保持高度的饭感;
⑥龙头板块其有便于大规模主流资金进出的流通容I;
⑦龙头板块的持续时问长，不会过早地分化和搜繁地切换。

# 龙兴股的种类
龙头股大致上有三大体系:一是以行业背景为基础的龙头;二是以概念背景为墓础的龙头;三是以地域背景为基础的龙头:另外还有包括多种背景的综合性龙头股。
①行业龙头
行业的龙头上市公司无疑有着强大的持续发展能力和巨大的市场吸引力，在各种类型的龙头股中，以行业的龙头股最多，近年来涨幅巨大的龙头股大多是以行业龙头股的身份出现的。
②概念龙头
概念性的龙头是二级市场中最容易上涨并对行情起落具有领导作用的有影响力股票，也可以称为板块龙头或领涨龙头。
③地域龙头
地域龙头是指同一地域构成的板块中所形成的龙头股。有的龙头股并非仅仅属于一种性质的龙头.而是属于多种性质的龙头股。
例如有的个股不仅仅是概念性的龙头股.它也是行业龙头股和地域性龙头股。这种综合性的龙头股往往更容易吸引主力资金的关注，行悄的爆发力也更强。而且更具有市场号召力和领导力，在各方资金的积极关注下.更容易掀起一飞冲天、天马行空般的龙头股行情。

# 龙头分类

是技术面、资金面、消息面合力的结果，三方面缺一不可。
第一类：重大政策。如2010年QE时的煤飞色舞。
第二类：新题材。如2012年的轨道交通龙一。
第三类：重大突发事件。
第四类：重组、摘冒。
第五类：送转。
最后一类，超级长线牛股。002456欧菲光

不难发现：
1 重大政策被市场认可后，主升周期最长，区间涨幅最大，可持续性最高。
2 新题材，
3 重大突发事件很多，但同样因规模和级别不够最终不被市场所认可。所以对于新题材、重大突发事件最好不要人为进行预测，市场永远是对的。
  当发现被市场认可后，选择龙一、二继续狙击操作，并作好快进快出的心理准备。
4 对于重组、摘帽、高送一般情况下难以形成板块效应。所以对于个股的分析上关键位置的关键板必须是高质量板。从而把偶然性降低的同时提高安全系数。

# 一字板总结

1 平均看，开板当日收益率低于次日收益率，行情不好，<开板当日可能大跌>

2 一字开板后的3种走势：
  a T字开板后，继续连板。
  b 开板后上午封板<资金净流入>，有龙头气象。(曙光股份、财信发展)。关键在大换手中判断出是否资金净流出，还是换庄。
  c 开板后高位换手<资金净流出>，大换手有<承接能力>，将在下降到位后反弹
  d 开板后高位换手<资金净流出>，<低换手>无承接能力，将在下降到位后沉寂

3 反弹计算公式

`
下跌幅度 = liu*资本净流出*(100/前两日平均换手) / (总股本*最高均价)
止跌价格 = 最高均价/(1+下跌幅度)

yellow=         # 均价
net_curr=       # 净流出
tov=            # 两日平均换手，代表筹码承接能力
liu=            # >5%的流通总和/33，与资金对价格影响力成正比
shares=         # 流通股本，单位：亿
yellow/(1 + liu*net_curr*(100/tov)/(shares*yellow))
`

注：换手代表筹码承接能力
    最后价格会有1%~3%的误差，可分仓以保稳健
    资本净流出，以龙虎榜为准

下面几个例子，几乎分毫无差

`
兔宝宝    2015-11-03 19.52/(1+3.35*(100/16.7)/(4.7*19.52)) = 16.01
隆华节能  2015-10-29 21.11/(1+1.39*(100/17.8)/(2.31*21.11))= 18.196 
联建光电  2015-12-09 31.94/(1+(.89+.55)*(100/12.9)/(2.03*31.94))

          48.41/(1+(.56)*(100/7.8)/(2.22*48.41))
凤凰股份  9.94/(1+.27*(100/6.85)/(7.41*9.94))
`

4 需要注意的几个问题
  a 流通因子：第一大流通股东占30%时为正常，占60%时则要x2，如凤凰股份
  b 误差计算：下跌幅度~=5%，取.5%，~=10%，取1%
